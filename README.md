# Robot code for Toadstool2026

Specs of Toadstool:

* Swerve: Redesigned MK2's with cim motors, and the AS5600 for the encoder.
* Body: 15.5"x15.5" without bumper, 22"x22" with bumpers.
* Network:
* Power: PDP 1.0
* Control: RoboRIO 2.0
* Motor controllers: Victor SPX
